# abdolkarim-bravi.github.io

این ریپو وب‌سایت شخصی عبدالکریم براویی را روی **GitHub Pages** میزبانی می‌کند.

## فایل‌های مهم
- `index.html` – صفحهٔ اصلی (واکنش‌گرا، RTL/LTR، ایمیل ضداسپم)
- `favicon.svg` – آیکون سایت
- `404.html` – صفحهٔ خطای ۴۰۴
- `robots.txt` – راهنمای ربات‌های جست‌وجو
- `sitemap.xml` – نقشهٔ سایت برای SEO
- `manifest.webmanifest` – تنظیمات PWA

## دامنه سفارشی (اختیاری)
اگر دامنهٔ اختصاصی دارید، فایل `CNAME` را بسازید و فقط نام دامنه را داخلش بنویسید.
مثال: `karimbravi.ir`
